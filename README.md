# 프로젝트 개요
- 시각장애인의 이동권 향상을 위해, 컴퓨터 비전 기술을 통해 전방의 지형지물 및 장애물을 안내하여 안전하게 보행하실 수 있도록 돕는 프로젝트

## 주요기능
- 바닥지형 안내
- 횡단보도 건너기 보조
- 장애물 충돌 위협 감지
